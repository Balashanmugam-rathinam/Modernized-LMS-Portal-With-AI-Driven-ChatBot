# EduSphere LMS

EduSphere is a modern, responsive Learning Management System (LMS) web portal designed to streamline educational processes for students and teachers. It features role-based access control, real-time attendance tracking, academic assessment analytics, and an integrated AI-powered assistant using Google Gemini.

## üåü Features

### üîê Authentication & Roles
- *Secure Login/Registration*: Role-specific flows for Students and Teachers.
- *Student Profile*: Captures detailed demographics, course info, and guardian details.
- *Roles*: 
  - *Student*: Access to learning resources, attendance scanning, and results.
  - *Teacher*: Class management, grading, and content verification.
  - *Admin*: System oversight (implied).

### üéì Student Portal
- *Dashboard*: At-a-glance view of CGPA, attendance percentage, and upcoming deadlines.
- *Attendance Scanner*: Enter a unique session code generated by the teacher to mark presence.
- *Assessment Analytics*: Visual charts (Bar/Pie) displaying semester-wise performance.
- *Notes Hub*: Upload and share study materials. Public notes require teacher approval.
- *Quiz Portal*: Gamified MCQ quizzes with instant feedback.
- *Certificates*: Digital repository for academic and co-curricular achievements.

### üë©‚Äçüè´ Teacher Portal
- *Session Management*: Generate dynamic, time-bound codes (and QR placeholders) for class attendance.
- *Gradebook*: View and edit student marks for continuous assessments and end-semester exams.
- *Content Moderation*: Review and approve/reject notes uploaded by students.
- *Class Insights*: Dashboard overview of active sessions and class average attendance.

### ü§ñ AI Chatbot (Gemini Powered)
- *Context-Aware Assistance*: The chatbot knows the user's name and role.
- *Intelligent Routing*: Guides users to specific modules (e.g., "Where are my marks?" -> Assessments View).
- *General Knowledge*: Answers educational queries using the Google Gemini 2.5 Flash model.

## üöÄ Getting Started

### Prerequisites
- A modern web browser.
- A Google Gemini API Key.

### Installation

1. *Clone the repository* (or download the files).
2. *Configure API Key*:
   Open services/geminiService.ts and replace process.env.API_KEY with your actual key, or ensure your build environment injects it.
   typescript
   const apiKey = process.env.API_KEY || 'YOUR_ACTUAL_API_KEY';
   
3. *Run the Application*:
   Since this project uses ES Modules and importmap, you can serve the root directory using any static file server.
   
   Using Python:
   bash
   python3 -m http.server 8000
   
   Using Node (http-server):
   bash
   npx http-server .
   

4. *Access*: Open http://localhost:8000 in your browser.

## üìÇ Project Structure

- **services/geminiService.ts**: Wrapper for Google Gemini API interactions.
- **constants.ts**: Mock data for initial application state.

## üìù Usage Guide

### For Students
1. *Register* as a new student.
2. Go to *Attendance* to enter the code provided by your teacher.
3. Check *Assessments* to see your grades.
4. Upload PDFs/Images in *Notes* (visible to others after approval).

### For Teachers
1. *Login* (Default: teacher@edusphere.com / password).
2. Go to *Attendance* to create a new session and display the code.
3. Go to *Assessments* to edit grades.
4. Go to *Notes* to approve pending student uploads.
